健康檢查

這一頁是給維護者看的，不是主入口。它的工作很單純：確認網站能不能正常打開、主要頁面在不在、JSON 有沒有壞掉、目前資料能不能作為公開展示版使用。

這頁要怎麼讀

先看整體狀態是不是 ok，再看有沒有 missing files、invalid JSON 或 warnings。如果這些檢查都正常，就代表網站本身適合拿來公開展示。

白話摘要

如果這一頁顯示正常，代表網站主要頁面存在、本地 JSON 檔案格式正確，而且目前資料站可以維持公開展示版的 prototype dashboard 狀態。這不代表正式資料已全量接入，只代表網站本身運作正常，並持續維持 no live crawler、no source_url requests、no personal data、no private complaint full text、no auto publish，且 approved_for_crawling 不自動設定。
