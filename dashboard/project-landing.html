PROJECT OVERVIEW

專案說明

嘉義市城市問題儀表板，是一個把公開資料、原型資料與地方議題整理成 KPI、議題排行、熱點地圖與行動建議的城市治理資料產品。這一頁用白話說明目前能看什麼、資料狀態如何、下一步要補什麼。

進入城市儀表板查看城市熱點地圖查看資料來源
目前狀態：這是 prototype dashboard，不是正式全量資料庫。正式資料接入前，不啟動 live crawler、不對 source_url 發出程式請求、不抓個資、不自動發布，也不把任何資料自動標成 approved_for_crawling。

這個網站要解決什麼問題

public purpose

看懂城市問題

把交通、停車、道路、環境、通學安全與市場商圈等問題整理成可看懂的分類與熱點。

追蹤問題位置

用城市熱點地圖呈現問題集中在哪些地點，讓討論從感覺變成可追蹤清單。

轉成改善行動

每個熱點都要對應主管機關、問題類型與下一步改善方向，而不是只做漂亮圖表。
