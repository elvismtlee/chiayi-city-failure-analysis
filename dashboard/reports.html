城市週報

把每週城市問題、熱點變化、資料狀態與下一步行動整理成可閱讀的摘要。這裡先提供 prototype weekly report 格式，正式資料接入後會逐週更新。

回城市儀表板 看熱點地圖 看資料來源
目前狀態：城市週報仍是 prototype dashboard 的一部分，不是正式全量資料報告。正式資料接入前，no live crawler、no source_url requests、no personal data、approved_for_crawling 不自動設定。

本週城市觀察

把首頁數字、熱點與議題排行整理成一週一看的摘要，讓沒有天天看 dashboard 的人也能快速理解重點。

prototype report
焦點 1

交通停車仍是第一優先

目前 prototype 指標顯示，交通與停車問題是最需要優先追蹤的城市問題類型。

焦點 2

市場商圈需要整合管理

市場周邊同時涉及環境、卸貨、通行與商圈動線，適合做跨局處改善清單。

焦點 3

通學安全可先從熱點做起

學校周邊的接送區、人行空間與路口安全，適合先用地點型專案追蹤。

週報會包含什麼

這一頁不是華麗包裝，而是固定用同一套欄位說清楚本週重點。

report structure

每週摘要

  • 本週最需要注意的城市問題。
  • 新增或升高的議題類型。
  • 熱點地圖與行動建議。

資料檢查

  • 資料來源是否更新。
  • JSON 是否可正常讀取。
  • prototype data 是否仍需標示限制。
