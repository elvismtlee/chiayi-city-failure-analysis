這頁在幫你看什麼

這一頁不是資料抓取頁，也不是內部流程首頁。它的作用是把目前可展示的原型資料、已盤點的官方資料來源，以及正式資料還沒全量接入的原因，先用白話講清楚。

資料透明原則

先把資料邊界說清楚，避免把原型數據誤認成正式全量資料。

public-readable
  • 目前首頁和地圖頁先使用 prototype dashboard data，不是正式全量資料。
  • 正式資料接入前不啟動 live crawler，也不對 source_url 發出程式請求。
  • 不抓個資、不抓私人陳情全文、不自動發布。
  • approved_for_crawling 不自動設定，所有正式接入都需要後續人工與技術確認。
prototype dashboard no live crawler no source_url requests no personal data approved_for_crawling 不自動設定

儀表板摘要

dashboard_summary.json

用來呈現原型案件數、質詢紀錄數、城市熱點與最大議題，是首頁 KPI 的基礎。

城市熱點資料

hotspots.jsonhotspots.geojson

issue_trends.json

把地方議題依時間窗與分類整理成趨勢資料，用來支撐首頁議題排行。
