# 嘉義城市故障週報規格 v1

## 目的

每週自動生成「嘉義城市故障週報」，作為：

- dashboard 更新
- 社群貼文
- 志工簡報
- 媒體說明
- 政見研究

## 資料來源

- crawler metadata
- issue ranking
- hotspot analysis
- AI summary

## 輸出格式

### HTML

```text
reports/html/*.html
```

### Markdown

```text
reports/markdown/*.md
```

### JSON

```text
reports/json/*.json
```

## 週報內容

### 1. 本週摘要

例如：

- 本週交通與停車問題持續增加
- 市場周邊垃圾與卸貨動線值得關注
- 學校通學安全議題增加

### 2. 熱門議題排行

- 停車
- 道路
- 人行
- 路燈
- 環境

### 3. 熱點地圖

- 文化路夜市
- 市場周邊
- 學校周邊

### 4. 議員質詢摘要

- 哪些議員關注交通
- 哪些議員關注文化
- 哪些議員關注教育

### 5. 局處回應分析

- 交通處
- 工務處
- 環保局

### 6. 建議行動

- 通學安全改善
- 夜市停車規劃
- 路燈巡檢

## 目標

把「資料」轉成：

- 可理解
- 可視覺化
- 可行動
- 可溝通

的城市治理資訊。
