# 資料表 Schema

## 1999陳情案件

| 欄位 | 說明 |
|---|---|
| case_id | 案件編號 |
| received_date | 收件日期 |
| district | 行政區 |
| village | 里別 |
| road_or_place | 路段或地點 |
| issue_category_lv1 | 第一層議題分類 |
| issue_category_lv2 | 第二層議題分類 |
| responsible_agency | 承辦局處 |
| status | 處理狀態 |
| processing_days | 處理天數 |
| source_url | 原始來源 |

## 議員質詢

| 欄位 | 說明 |
|---|---|
| question_id | 質詢編號 |
| meeting_date | 會議日期 |
| councilor_name | 議員姓名 |
| topic_category_lv1 | 第一層議題 |
| topic_category_lv2 | 第二層議題 |
| department | 對應局處 |
| question_summary | 質詢摘要 |
| video_url | 影音來源 |
| minutes_url | 會議紀錄來源 |
