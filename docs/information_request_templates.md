# 資訊公開與資料申請範本 v1

本文件整理未來若需要向機關申請資料時，可使用的文字範本。

---

## 目的

若部分 1999 / 陳情資料沒有公開下載，可透過正式管道詢問是否有：

- 統計資料
- 去識別化資料
- 開放資料集
- 年度報告
- 可公開引用的彙整資料

---

## 詢問信範本：1999 / 陳情統計資料

```text
您好：

我正在進行嘉義市城市治理與地方生活議題資料整理，希望了解嘉義市政府是否有提供 1999、市政信箱或陳情案件之公開統計資料或去識別化資料。

想請教是否有以下資料可供查詢或申請：

1. 近年各類陳情案件統計
2. 各行政區案件數統計
3. 各局處承辦案件統計
4. 案件分類與處理狀態統計
5. 可公開使用之去識別化資料集

本研究用途為城市治理、公共服務與地方議題分析，不會公開個人姓名、電話、email、完整地址等個人資料。

若已有公開資料平台或年度統計報告，也請協助提供連結。

感謝協助。
```

---

## 詢問信範本：開放資料授權與使用限制

```text
您好：

想請教貴單位公開資料或網頁資料之再利用限制。若資料已公開於官方網站，是否可於保留來源連結、更新日期與資料限制說明的情況下，用於非商業之城市治理資料分析與視覺化展示？

若有授權條款、引用格式或使用限制，煩請提供相關說明。

感謝協助。
```

---

## 詢問信範本：地理資料 / 里界資料

```text
您好：

我想詢問嘉義市行政區、里界、道路或公共設施相關 GIS / GeoJSON / Shapefile 資料是否有公開下載來源。

資料用途為城市治理資料視覺化、熱點分析與地方議題研究。若已有公開資料平台連結，煩請協助提供。

感謝協助。
```

---

## 注意事項

1. 只申請可公開或去識別化資料。
2. 不要求個人資料。
3. 用途說明要清楚。
4. 取得資料後仍要保留來源與限制。
5. 若資料不能公開，只能用統計摘要或不使用。
