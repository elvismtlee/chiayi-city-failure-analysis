# 嘉義在地名稱與用語規範 v1

本文件定義「嘉義城市故障分析資料庫」在 dashboard、週報、AI 摘要、社群文案與現場行動中應使用的嘉義在地名稱與語氣。

---

## 目的

避免使用外地觀光式、資料庫式或不符合嘉義人日常說法的名稱。

城市治理資料若要讓在地市民信任，地名與語氣必須符合地方生活經驗。

---

## 基本原則

1. 優先使用嘉義人日常會說的地名。
2. 若正式地名與日常地名不同，對外文案優先用日常地名，資料欄位可保留正式名稱。
3. 不把商圈、街區、路段一律寫成觀光景點。
4. 不把所有晚間人潮聚集區都稱為夜市。
5. 對外文案要像嘉義人說話，不要像外地觀光簡介。

---

## 文化路用語規範

### 建議用法

| 使用情境 | 建議用語 |
|---|---|
| 一般口語 / 社群 | 文化路 |
| dashboard 熱點 | 文化路商圈 |
| 週報 / 分析 | 文化路商圈周邊 |
| 地點標準化 | 文化路商圈 |
| 路段分析 | 文化路部分路段 |

### 避免用法

```text
文化路夜市
```

原因：

```text
嘉義在地人通常不會把該區稱為「文化路夜市」，較常說「文化路」或「文化路商圈」。
```

---

## 建議地名寫法

| 不建議 | 建議 |
|---|---|
| 文化路夜市 | 文化路 / 文化路商圈 |
| 火車站商圈入口 | 嘉義火車站前站周邊 |
| 嘉義市中心夜市區 | 文化路商圈周邊 |
| 老街區 | 舊市區 / 老城區 / 文化路周邊 |
| 市中心 | 舊市區 / 文化路周邊 / 中央廣場周邊 |

---

## dashboard 顯示原則

地點名稱應短、清楚、在地：

```text
文化路商圈
嘉義火車站前站
北興市場周邊
中央廣場周邊
民生南路部分路段
新民路部分路段
世賢路部分路段
```

避免：

```text
嘉義著名夜市觀光區
嘉義老街美食景點
嘉義市中心鬧區
```

---

## AI 摘要用語原則

AI 產生摘要時應使用：

```text
文化路商圈周邊出現交通、人行與商圈動線議題。
```

避免使用：

```text
文化路夜市人潮造成交通混亂。
```

原因：

1. 「夜市」不是在地常用名稱。
2. 「造成交通混亂」語氣過度定論。
3. 應改用可追蹤、可改善的中性語氣。

---

## 週報用語範例

建議：

```text
本週觀察文化路商圈周邊交通與人行動線議題，尖峰時段仍可列為後續追蹤重點。
```

避免：

```text
文化路夜市交通問題嚴重，證明市民不滿。
```

---

## 社群文案用語範例

建議：

```text
嘉義人熟悉的文化路，不只是吃東西、逛街的地方，也是很多人每天經過的生活路線。商圈周邊的停車、人行與動線問題，值得被好好整理與追蹤。
```

避免：

```text
嘉義文化路夜市是熱門景點，所以交通一定很亂。
```

---

## 未來新增地名時的檢查問題

新增地點前先問：

1. 嘉義人平常真的這樣稱呼嗎？
2. 這是正式地名、觀光名稱，還是在地口語？
3. dashboard 對外顯示是否太像觀光文案？
4. 是否可能讓在地人覺得不精準？
5. 是否需要保留別名欄位？

未來地點資料可保留：

```json

  "formal_name": "文化路商圈",

  "avoid_terms": ["文化路夜市"]

```

讓資料像嘉義人整理的，不像外地人用觀光印象寫的。
