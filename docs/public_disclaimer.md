```text
本頁為嘉義城市故障分析資料庫之原型展示。資料來自公開資訊、官方資料與樣本資料整理，僅供城市治理與地方議題研究參考，不代表完整民意調查或個人評價結論。
```

## 中版聲明

```text
嘉義城市故障分析中心以公開資料、官方資訊與地方議題資料為基礎，整理城市生活問題、熱點與趨勢。現階段部分資料仍為 prototype / sample data，正式資料上線後，將標示資料來源、更新時間與資料限制。分析結果僅供城市治理、政策研究與地方服務參考，不代表完整民意調查，也不作為個人評價結論。
```

## 長版聲明

```text
本平台為「嘉義市 12 年城市故障分析資料庫」之公開展示原型，目的在於協助理解嘉義市生活議題、公共服務需求與城市治理挑戰。平台資料來源包括官方公開資訊、議會資料、政府開放資料、地方議題整理與樣本資料。所有分析結果皆應回到原始資料查證，且不得視為完整民意調查、支持度調查或個別人員評價結論。正式資料上線後，本平台將盡可能標示資料來源、更新時間、處理方法與資料限制。
```
