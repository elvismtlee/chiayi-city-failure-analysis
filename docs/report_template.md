## 一、本週摘要

目標：用 150 - 300 字說明本週最重要的城市觀察。

寫法：

```text
本週資料顯示，＿＿＿仍是主要觀察議題。＿＿＿區域出現較高關注度，可能與＿＿＿有關。由於目前資料仍在整理階段，本週報僅作為城市治理與地方議題研究參考。
```

## 五、議員質詢摘要

寫法原則：

1. 只描述公開資料中出現的議題。
2. 不推論議員是否關心或不關心。
3. 資料不足時使用「公開資料中較少出現」而非「未關心」。

範例：

```text
公開質詢 metadata 顯示，本期資料中交通、道路與市場周邊議題較常出現。由於目前尚未完成全文解析，後續仍需結合會議紀錄與影音逐步補強。
```

## 六、局處負荷觀察

寫法原則：

1. 案件量高不等於效率差。
2. 只能稱為負荷觀察，不做責任判定。
3. 需標示資料限制。

範例：

```text
以原型資料觀察，交通與工務相關議題可能形成較高案件負荷。此結果僅代表資料中的議題分布，不能直接解讀為局處績效結論。
```

## 七、建議行動

建議分成：

### 短期

- 現場盤點
- 拍照記錄
- 建立地點清單

### 中期

- 與局處溝通
- 追蹤處理狀態
- 建立改善前後對照

### 長期

- 轉為制度性改善
- 建立每月資料公開
- 建立跨局處協作流程

---

## 八、資料來源與限制

固定聲明：

```text
本週報資料來自公開資訊、官方資料與原型資料整理。分析結果為城市治理與地方議題研究用途，不代表完整民意調查，也不作為個人評價結論。正式引用前，仍應回到原始來源查證。
```

## 語氣原則

1. 溫和、堅定、具體。
2. 工程師邏輯，但不要艱澀。
3. 有嘉義在地感。
4. 不抹黑、不造謠、不人身攻擊。
5. 把問題轉成可追蹤、可改善的行動。
6. 地名應符合嘉義在地稱呼，參考 `docs/local_terminology_style_guide.md`。
