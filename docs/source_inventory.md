# 官方資料源盤點

更新日期：2026-05-21

## 1. 嘉義市議會全球資訊網

首頁：<https://www.cycc.gov.tw/Default.aspx>

可用入口：

- 會議紀錄檢索
- 議決案檢索
- 本屆議事錄
- 議員質詢影音專區
- 議事日程表
- 新聞資料
- 公告訊息

優先級：最高

用途：

- 建立議員質詢資料表
- 建立會議紀錄資料表
- 建立議題 / 局處 / 議員對照
- 形成投訴 vs 質詢對照分析

## 2. 嘉義市政府

首頁：<https://www.chiayi.gov.tw/>

待盤點：

- 1999 服務資訊
- 線上陳情系統
- 市政信箱
- 各局處公告
- 施政成果
- 統計資訊

優先級：高

注意：1999 或陳情逐案明細可能未完整公開，需用公開資料、資訊公開申請或統計報告補足。

## 3. 政府資料開放平台

首頁：<https://data.gov.tw/>

搜尋關鍵字：

- 嘉義市 1999
- 嘉義市 陳情
- 嘉義市 交通
- 嘉義市 道路
- 嘉義市 環保
- 嘉義市 路燈

優先級：高

用途：

- 補足 CSV / JSON / API 資料
- 與議會質詢資料交叉比對

## 4. 地方新聞與公開資訊

用途：

- 議題補充
- 年度事件脈絡
- 熱點驗證

注意：新聞只能作為輔助資料，正式分析仍以官方資料為主。
