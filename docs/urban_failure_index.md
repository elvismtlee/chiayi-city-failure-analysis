# 城市故障指數 Urban Failure Index

本文件定義「城市故障指數」的概念。

目的：

1. 找出長期未改善問題
2. 找出高熱度生活議題
3. 找出市民反映多但政治關注低的區域
4. 作為儀表板與週報核心指標

---

## 基本公式

```text
城市故障指數 =
案件量 × 重複率 × 處理天數 × 質詢落差
```

---

## 指標來源

| 指標 | 來源 |
|---|---|
| 案件量 | 1999 / 陳情 |
| 重複率 | 同地點重複案件 |
| 處理天數 | closed_date - received_date |
| 質詢落差 | 市民投訴 vs 議員質詢 |

---

## 可分析層級

- 行政區
- 里別
- 路段
- 商圈
- 市場
- 學校周邊
- 局處
- 議題分類
