# 嘉義城市故障分類標準 v1

本文件定義「嘉義市 12 年城市故障分析資料庫」的議題分類架構。

分類目的：

1. 將市民陳情、1999、議員質詢、新聞、地方回報統一分類
2. 方便後續 AI 自動分類
3. 方便 dashboard 統計
4. 方便產生地方議題、政見、短影音腳本

---

## Lv1 分類

| Lv1 | 說明 |
|---|---|
| 交通 | 停車、號誌、車流、公共運輸、學校周邊交通 |
| 道路 | 路平、坑洞、施工、排水孔蓋、路面標線 |
| 人行 | 人行道、騎樓、斑馬線、無障礙、行人安全 |
| 環境 | 垃圾、異味、空污、噪音、髒亂、回收 |
| 公共安全 | 路燈、治安死角、危險路口、夜間照明 |
| 水利排水 | 水溝、淹水、側溝、排水不良、積水 |
| 市場商圈 | 傳統市場、夜市、商圈動線、攤販、停車 |
| 公園休憩 | 公園、遊具、綠地、座椅、廁所、運動空間 |
| 教育學區 | 校園周邊、通學安全、學校設施、親子動線 |
| 社福高齡 | 高齡友善、長照、無障礙、社區關懷 |
| 文化觀光 | 文化活動、觀光動線、歷史街區、城市品牌 |
| 行政服務 | 局處效率、陳情處理、資訊公開、跨局處協調 |
| 其他 | 無法歸類或資料不足 |

## Lv2 / Lv3 建議分類

### 交通

| Lv2 | Lv3 關鍵問題 |
|---|---|
| 停車 | 停車格不足、違停、停車場動線、卸貨停靠 |
| 號誌 | 紅綠燈秒差、行人號誌、號誌故障 |
| 車流 | 塞車、轉彎動線、單行道、路口衝突 |
| 公車 | 班次、站牌、候車環境、轉乘 |
| 通學交通 | 學校周邊接送、斑馬線、導護安全 |

### 道路

| Lv2 | Lv3 關鍵問題 |
|---|---|
| 路平 | 坑洞、凹陷、補丁、龜裂 |
| 施工 | 施工品質、施工期太長、標示不足 |
| 標線 | 標線模糊、車道線不清、斑馬線退色 |
| 孔蓋 | 人孔蓋高低差、噪音、危險 |

### 人行

| Lv2 | Lv3 關鍵問題 |
|---|---|
| 人行道 | 斷點、寬度不足、障礙物、破損 |
| 騎樓 | 占用、通行不順、無障礙落差 |
| 行人穿越 | 斑馬線、庇護島、行人號誌 |
| 無障礙 | 輪椅、嬰兒車、高齡者通行 |

### 環境

| Lv2 | Lv3 關鍵問題 |
|---|---|
| 垃圾 | 清運、亂丟、廢棄物、垃圾桶不足 |
| 異味 | 水溝味、餐飲油煙、廢氣 |
| 噪音 | 交通噪音、工地噪音、商業噪音 |
| 髒亂 | 空地雜草、街角髒亂、牆面髒污 |

### 公共安全

| Lv2 | Lv3 關鍵問題 |
|---|---|
| 路燈照明 | 照明不足、故障、死角 |
| 危險路口 | 視線不良、車速快、事故熱點 |
| 治安死角 | 夜間安全、監視器、巡邏需求 |

## AI 分類規則

AI 分類時應依序判斷：

1. 是否有明確地點
2. 是否有明確局處
3. 是否有明確關鍵字
4. 是否能對應 Lv1 / Lv2 / Lv3
5. 若信心不足，先標為「其他」並加入 notes

---

## 優先分析議題

西區第一階段建議優先分析：

1. 停車
2. 市場周邊交通
3. 道路坑洞與路平
4. 人行道與騎樓通行
5. 路燈照明
6. 垃圾與環境髒亂
7. 學校周邊通學安全
